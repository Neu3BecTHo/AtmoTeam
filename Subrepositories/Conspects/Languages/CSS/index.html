Свойства и значения CSS

color

Меняет цвет шрифта. Принимает как английские названия (red), так и RGB, RGBA и т.п.

width

Позволяет задать ширину нужному селектору. Принимает значения в пикселах, процентах и других значениях (vh, vw и т.п.)

height

Позволяет задать высоту нужному элементу. Принимает значения в пикселах, процентах и других значениях (vh, vw и т.п.)

text-align

Позволяет изменить позиционирование текста. Принимает значения right, left, justify, center, start, end и другие некоторые значения.

font-size

Позволяет задать тексту оперделённую ширину. Принимает значения в пикселах, процентах и других значениях (vh, vw и т.п.).

display

Позволяет задать селектору его отображение. Значений много, но самые популярные из них - block, inline, inline-block, flex, grid.

position

Определяет расположение селектора. Принимает значения static, relative, absolute и sticky.

background

Включает в себя разные значения для фона. Можно задавать цвет, фоновое изображение, его повторение и т.п.
