Молекулярная физика

Постоянные значения

NА = 6,02 * 1023 моль-1;

R = 8,31Дж⁄моль * К;

k = 1,38 * 10-23 Дж⁄К;

Формула концентрации молекул

n = N⁄V

n - концентрация [м-3];
N - количество молекул;
V - объём [м3];

Формула плотности

p = m⁄V

p - плотность вещества [кг⁄м3];
m - масса вещества [кг];
V - объём [м3];

Формула относительной молекулярной массы

Mr = m0⁄1⁄12mс

Mr - относительная молекулярная масса;
m0 - масса одной молекулы [кг];
mс - масса атома углерода [кг];

Формула количества вещества (количества молей)

v = m⁄M

v - количество вещества (количество молей) [моль]
m - масса вещества [кг]
M - молярная масса [кг⁄моль]

Формулы массы одной молекулы

  • m0 = m⁄N

    • m0 - масса одной молекулы [кг];
    m - масса вещества [кг];
    N - количество молекул;

  • m0 = M⁄NА

    • m0 - масса одной молекулы [кг];
    M - молярная масса [кг⁄моль];
    NА - число Авогадро [моль-1];

  • m0 = p⁄n

    • m0 - масса одной молекулы [кг];
    p - плотность вещества [кг⁄м3];
    n - концентрация молекул [м-3];

Формулы количества молекул

  • N = nV

    • N - количество молекул;
    n - концентрация молекул [м-3];
    V - объём [м3];

  • N = vNА

    • N - количество молекул;
    v - количество вещества (количество молей) [моль];
    NА - число Авогадро;

  • N = m⁄m0

    • N - количество молекул;
    m - масса вещества [кг];
    m0 - масса одной молекулы [кг];

Формулы средней квадратичной скорости молекул

  • ⊽ = √3RT⁄M

    • ⊽ - средняя квадратическая скорость молекулы [м⁄с];
    R - молярная газовая постоянная;
    T - абсолютная температура [К];
    M - молярная масса [кг⁄моль];

  • ⊽ = √3kT⁄m0

    ⊽ - средняя квадратичная скорость молекул [м⁄с];
    k - постоянная Больцмана;
    T - абсолютная температура [К];
    m0 - масса одной молекулы [кг];

    • k = R⁄NА

      • k - постоянная Больцмана;
      R - молярная газовая постоянная;
      NА - число Авогадро [моль⁄-1];
