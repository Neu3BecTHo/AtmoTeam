Введение

На данной странице будут тесты многих тегов. Тут я надеюсь проверить и протестировать многие теги, их свойства, функции и т.д. Возможно, по окончании этой странички я смогу поделиться ею с друзьями и просто с людьми, изучающими html, css и, возможно, javascript.

Возможны ошибки и неточности. Ваши ожидания могут не оправдаться, а опытные web-разработчики могут не найти тут чего-то нового. Руководство будет написано для новичков.

По возможности введение будет дополняться.

IDE

IDE является специальным редактором, который базируется на каком-то из языков программирования. Например, есть редактор. Он позволяет писать программы, сайты и даже игры (Конечно же, код). Так же в нем есть полезные функции по типу автозаполнения, исправления ошибок и подсветкой текста, что очень помогает определиться, правильно ли написан кусочек кода. Как по мне, то Sublime самый быстрый и лёгкий из всех редакторов. Он написан, как помню, на c++ и из-за этого редактор получил быстроту.
